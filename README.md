# SimCH Engine

## Vision

SimCH Engine adalah Autonomous AI Simulation Engine untuk membangun dunia yang hidup.

Dalam SimCH Engine:

- Player bukan pengendali NPC.
- Setiap Agent memiliki tujuan.
- Setiap Agent dapat mengambil keputusan.
- Dunia terus berjalan meskipun player tidak melakukan apa pun.
